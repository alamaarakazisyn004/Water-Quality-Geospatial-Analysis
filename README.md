# Water Quality Geospatial Analysis & Risk Modeling

## Project Overview
This project analyzes multi-year water quality data (2018–2022) to identify environmental trends and predict pollution risk levels using geospatial techniques and machine learning.

---

## Objective
- Perform geospatial trend analysis
- Identify pollution patterns
- Build environmental risk prediction model

---

## Dataset
- Source: National Water Quality Monitoring Council (NWQMC)
- Features:
  - pH
  - BOD (Biochemical Oxygen Demand)
  - DO (Dissolved Oxygen)
  - State-wise data

---

## Tools & Technologies
- Python
- Pandas
- GeoPandas
- Folium
- Scikit-learn

---

## Key Steps
- Data cleaning and merging (2018–2022)
- Feature engineering
- Geospatial mapping
- Trend analysis
- Risk classification
- Machine learning model (Random Forest)

---

## Results
- Identified high-risk regions based on pollution levels
- Created interactive maps for visualization
- Achieved good prediction accuracy using ML

---

## Future Scope
- Use real GPS coordinates for better accuracy
- Integrate real-time water monitoring data
- Deploy as a web dashboard
